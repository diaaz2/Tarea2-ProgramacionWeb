Proyectos Realizados

Proyecto 1: Proyecto final programacion I

El objetivo de este proyecto era utilizar las bases de datos, consistio en una combinacion de varios CRUD y formularios de registro e inicio de sesion, en los que se guardaran usuarios que se registren, y luego de iniciar sesion permitiera interactuar con los CRUD.

Proyecto 2: Manejar texto

Este proyecto consiste en un programa que permite realizar algunas acciones con el texto, entre estas acciones estan invertir el texto, eliminar las vocales de una palabra u oracion, contar las vocales, contar las consonantes o contar la cantidad de palabras de un texto.

Proyecto 3: Calculadora

El programa presenta un menu en el que podemos elegir la operacion que deseamos realizar, entre las que se encuentran suma, resta, multiplicacion y division.
